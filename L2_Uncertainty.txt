# Terms

    *Probability*- P(w), must be between and include 0 and 1
        The sigma, summation, of it must =1
        Probabilities might be equal, not just sum of probabilities

    *Unconditional Probability*-
        Degree of belief in a proposition w, in the absence of any other evidence

    *Conditional Probability*-
        Degree of belief in a proposition w, given some evidence that's revealed
        P(a | b), a is what we want to be true, b is the evidence given
        Calculation: P(a | b) = P(a ^ b) / P(b)    interchangeable

    *Random Variable*-
        Variable with a domain of possible values it can assume
            Uses probability in determining the decision, uses <>, vector as in order

    *Independence*- The knowledge of one event doesn't influence the probability of other events
        Ex. P(a ^ b) = P(a)P(b)    not P(b | a)

    *Joint Probability*-
        Probability distribution of each possible combination from the random variables
        aP(., .) represents |  P(. | .) = P(., .) / p(.)
        a uses as multiplier, if . + . doesn't == 1, then use a to multiply

# Rules

    *Bayes' Rule*-

        if  P(a ^ b) = P(a)P(b | a), and
            P(a ^ b) = P(b)P(a | b)

        then   P(a)P(b | a) = P(b)P(a | b)

        which P(b | a) = P(b)P(a | b) / P(a)

    *Negation*-

        P(¬a | a) = 1 - P(a)

    *Inclusion-Exclusion*-

        P(a ∨ b) = P(a) + P(b) - P(a ^ b)

    *Marginalization*-

        P(a) = P(a, b) + P(a, ¬b), or
        P(x = x_) = summation (j) P(Y = x_, Y = yj)

    *Conditioning*-

        P(a) = P(a|b)P(b) + P(a|¬b)P(¬b) or
        P(x = x_) = summation (j) P(X = x_| Y = yj) P(Y = yj)

# Bayesian Network

    Data structure that represents the *dependencies* among random variables
        Directed graph
        Each node is a random variable
        Arrow from x to y means x is a parent of y
            P(X| parents(x))

    Approximate Inference- Maximize accuracy over time without producing exact inference

        Sampling- Enumerating all possible variables and picking one
            As process continues, we use each previous variable
                Prefer using conditional
                Reject other variables, randomize

        Likelihood Weighting-
            Fix variables' values, so the process will skip them
            From Bayesian Network, sample from conditional probability
            Weight sample by its likelihood, frequency

# Markov Assumption/Model

    - Assumption that current state depends on a finite fixed number of previous states, like only yesterday or last week
      - With a starting distribution

    Markov Chain- Sequence of random variables' distributions that follow the Markov assumption
        Contains hidden state, but resolved from observation

    Hidden Markov Model- Markov model with hidden states that generate some observation event

    Sensor Model/Emission Probabilities- The state emits an observation by emission

    Sensor Markov Assumption- Evidence variable depends only on the corresponding state

    # Tasks

        Filtering- Calculate distribution for current state from starting state
        Prediction- From observation, calculate distribution for future state
        Smoothing- Calculate distribution for past state from starting state
        Most_Likely_Explanation- Calculate most likely sequence from starting state

# Inference

    Query X- Variable to compute
    Evidence E- Observed variables for event e
    Hidden Y- Non-evidence, Non-Query variable
    Goal: Calculate P(X| e)

    ex.     P(Appointment| light, no)
            = a P(Appointment, light, no)
            = a[ P(Appointment, light, no, on time)
              +  P(Appointment, light, no, delayed)]

    P(X| e) = a P(X, e) = a summation(Y) P(X,e,y)
    X = Query
    e = evidence
    y = ranges of hidden variables
    a = normalizes the result to 1
