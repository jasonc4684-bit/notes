Hill Climbing Algorithm-

    Hill Climbing - Start a state, change the state based on the value of neighboring state,
        maintain if it's opposite with goal, else continue
            Might potentially stopped by local maxima/ minima
            Also stuck by flat local maxima/ minima / shoulder

        Variant (but never create worse moves):
            Steepest ascent - choose the highest-valued neighbor
            Stochastic - choose randomly from higher-valued neighbors
            First-Choice - choose the first higher-valued neighbors

            Reduce being stuck (repeating):
                random-restart - conduct hill climbing multiple times
                local beam search - chooses the k highest-valued neighbors, starts more than 1 state

    Hill Climbing function ex:

        function hill_climbing (problem):
            current = initial state, either pick or random
            repeat:
                neighbor = highest/lowest valued neighbor of current, random if both are equal
                if neighbor not better than current:
                    return current
                current = neighbor

Simulated Annealing-

Arc consistency function-

    Single arc-

        function Revise (csp, X, Y): make X consistent to Y
            revised = false
            for x in X domain:
                if no y in Y domain satisfy constraint for (X, Y):
                    delete x from X domain
                    revised = true
            return revised

    entire arc-

        function AC-3(csp):
            queue = all arcs in csp
            while queue non-empty:
                (X,Y) = Dequeue(queue)
                if revised (csp, X, Y):
                    if size of X domain ==0:
                        return False
                    for each Z in X neighbors - {Y}: except Y
                        Enqueue(queue, (Z, X))
            return True

Backtracking Search (revised) - csp is not in the parameter, failure = None

    function Bt (assignment, csp):
        if assignment complete: return assignment:
        var = select_unassigned_var (assignment, csp)
        for value in domain-values(var, assignment, csp) or []:
            if value consistent with assignment:
                add {var = value} to assignment     ( assignment[var] = value )
                inferences = Inference (assignment, csp)
                if inferences != failure: add inferences to assignment
                result = Bt (assignment, csp)
                if result doesn't fail:
                    return result
            remove {var = value} and inferences from assignment
        return failure
